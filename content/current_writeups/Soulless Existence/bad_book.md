The book reads as Time Magic but when you can read the words in translation, it was written in a Void language. The Golem want the book to be destroyed, the Agent wants to escape after stealing it from the Vindale Vampire Library. It’s up to the Adventurers who gets to keep the book or if it is destroyed. The Void language in the book is for a summoning ritual of Void creatures.
